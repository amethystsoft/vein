# ``VeinSwiftUI``

Vein for use with SwiftUI.

## Overview

To use the engine components, simply add the import to your Swift files:

```swift
    import VeinSwiftUI
```

## Core Infrastructure

This package automatically re-exports and exposes the core algorithmic engines and identifier formats. 
Refer to the Vein documentation for deep-dive API specifications and to the VeinCore tutorials for general tutorials.
